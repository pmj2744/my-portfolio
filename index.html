About Me

저는 사용자가 보기 쉽고 일관성 있는 웹 화면을 만드는 것이 목표입니다.

이번 포트폴리오는 수업 시간에 배운 HTML 시맨틱 태그와 CSS 레이아웃 기법을 활용해 제작했습니다.
